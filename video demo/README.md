# 🎬 Video Demo

This folder contains the video demonstration for the **Book a Doctor** MERN Stack Project.

---

## 📽️ About the Demo

The video demo provides a complete walkthrough of the **Book a Doctor** web application, showcasing all major features and functionalities built using the MERN (MongoDB, Express.js, React.js, Node.js) stack.

---

## 🔍 What's Covered in the Demo

- **🏠 Home Page** – Overview of the landing page and navigation
- **👤 User Registration & Login** – Patient and admin authentication flow
- **🔎 Doctor Search** – Browse and filter doctors by specialty or location
- **📅 Appointment Booking** – Step-by-step appointment scheduling process
- **🗂️ Appointment Management** – View, manage, and cancel appointments
- **🩺 Admin Dashboard** – Admin panel for managing doctors and appointments
- **📱 Responsive Design** – Mobile and desktop UI walkthrough

---

## 🛠️ Tech Stack Demonstrated

| Technology | Role |
|---|---|
| **MongoDB** | Database |
